# Contrast Check (Fridge Combo #51)

A hyper-minimalist, browser-based WCAG 2.1 contrast validator. Design with accessibility in mind, instantly.

## 1 Tool / 1 Action
- **The Action:** Enter two colors and instantly verify if their contrast ratio meets the WCAG AA standard (4.5:1).

## Features
- **Dual-Size Preview:** Real-time visualization of both Headline (2.0rem) and Body text (1.0rem) to ensure readability across different hierarchy levels.
- **Smart Advice:** If the contrast fails, the tool detects the background brightness and suggests whether you should go darker or lighter.
- **Natural Language Support:** Accepts both Hex codes (e.g., `#333333`) and CSS color names (e.g., `white`, `royal务blue`).
- **iOS Optimized:** Custom CSS to prevent input text fading and system UI interference on Safari.
- **Privacy First:** No login, no tracking, no ads. Works entirely in your browser.

## Design System
- **Background:** `#ffffff`
- **Main Elements:** `#333333`
- **Border Radius:** `15px` (Mandatory for all UI elements)
- **Performance:** Lightweight vanilla JS with no external dependencies. Optimized for low-spec mobile and desktop devices.

## How to Use
1. Enter your **Background Color**.
2. Enter your **Text Color**.
3. Check the **PASS/FAIL** badge.
4. Follow the built-in advice if the contrast is insufficient.
